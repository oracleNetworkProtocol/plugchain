# PCPrice

### Request method
`GET`

### URL
`/server/v1/market/pc`

### Request example

```
/server/v1/market/pc
```


#### Request parameters
- Not

### Return value
- `code int` : Data returned successfully
- `data string` : Price information
- `message string` : Return information

#### Return to example
```json
{
  "code": 0,
  "data": "0.00277801",
  "message": "ok"
}
```
